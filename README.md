
# Project Title

Nous avons ici un procédé de localisation de numéro de téléphone. Le numéro est inséré dans le fichier myNumber et après l'éxécution du fichier numberLocation, nous avons un fichier location.html qui se crée et affiche la localisation du numéro recherché.




## Environment Variables

Pour executer ce projet, vous aurez besoin d'un environnement virtuel 

python -m venv venv
venv\Scripts\activate
## Installation

Installer le projet

```bash
    pip install phonenumbers
    pip install opencage
    pip install folium
```
